Getting Started with the PicoScope Support Toolbox

Welcome to the PicoScope Support Toolbox!

This guide will help you with using the classes and functions provided in this toolbox whether you are using the PicoScope Instrument Drivers or calling the driver Application Protocol Interface (API) functions directly from MATLAB.

The sections below provide a brief description of what is provided in the toolbox - use the links provided to view further information.

Contents

  • Classes
  • Functions

Classes

PicoScope Support Toolbox provides classes with defined constant properties that may be used when communicating with PicoScope hardware products, including status codes which are returned by functions in most PicoScope drivers.

Functions

PicoScope Support Toolbox provides functions that can assist with the following:

  • Converting between Analogue Digital Converter values and millivolts.
  • Graphical User Interface (GUI) components that can be used when collecting data using a PicoScope hardware product in streaming mode.
  • Importing an arbitrary waveform from a file.
  • Normalising vectors defining arbitrary waveforms.
